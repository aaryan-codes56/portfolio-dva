## 🏗 Setup & Installation

To run this portfolio locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/aaryan-codes56/portfolio-dva.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio-dva
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
